# termtube
Search and Watch Youtube videos from terminal
# Requirements
- [fzf](https://github.com/junegunn/fzf) (optional)
- [mpv](https://mpv.io/installation/) (optional)
- [python3.x](https://www.python.org/downloads/)

# Usage
## From you terminal :

Simply run :
```bash
    python3 main.y [option][search_query]
```
